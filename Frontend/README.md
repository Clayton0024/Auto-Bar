# Vite Application Setup

## Dependencies

React 18.2.0
Vite 5.0.8

Node v18.17.1
NPM 10.2.5

## Setup Environment

```bash

# Navigate to the project directory

cd Frontend

# Install dependencies

npm  install

# Basic Commands

npm run dev (start dev server)

npm run build (build for production)

npm run lint (linting)

npm run preview (Vite Preview)
```
